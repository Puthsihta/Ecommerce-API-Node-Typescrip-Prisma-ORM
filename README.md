## Before Start Project :

1. must have .env file
   ![DB Image](assets/connet_db.png)

2. connet db with prisma orm

- migration to create table : npx prisma migrate dev --name MIGRATIONNAME

3. start project : yarn start

## Prisma command :

- npx prisma studio
- npx prisma generate
